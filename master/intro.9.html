INTRO(9) Kernel Developer's Manual INTRO(9)

NAME

intro — Introduction to Keyronex kernel internals

DESCRIPTION

This section contains documentation related to the kernel internals of the kernel. It covers kernel facilities and functions that are relevant to developers of the kernel, drivers, and modules. These manual pages are intended to be the primary reference for kernel programming and a complement to in-source documentation. Below are presented sections for each high-level division of the kernel, containing links to individual subsystems and functions.
